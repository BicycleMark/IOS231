Before you begin

This is an iOS class. To complete the coding exercises, you will need a Xamarin.iOS development environment with either a Simulator or a device to run the exercise on. Please see the setup page if you need help installing the Xamarin development environment.

Overview

In this class you will learn how to control and customize the appearance of annotations on a map. You'll also learn how to add overlays, search for points of interest, and show directions.

Objectives

  1. Create custom annotations
  2. Group annotations with clusters
  3. Interact with annotations
  4. Search for points of interest
  5. Add overlays and get directions

Lecture

IOS231 - Customizing Maps and Location.

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Change the marker color
    Exercise
    In this exercise, we will update the marker color based on data provided in a custom annotation.
  2. Create a custom annotation
    Exercise
    In this exercise, we will create a custom annotation view and display it on the map.
  3. Enable cluster support
    Exercise
    In this exercise, we will group annotations together using clusters.
  4. Add behavior to the annotations
    Exercise
    In this exercise, we will respond to user taps on the map annotations.
  5. Search for Points of Interest
    Exercise
    In this exercise, we will search for points of interest within the region displayed on the map.
  6. Add overlays and display directions
    Exercise
    In this exercise, we will add overlays to the map, both to highlight a geographic region and to show directions to points of interest.
